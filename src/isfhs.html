Information System on Foods and Housing Services

ISFHS is an information system about Foods and Housing around NEUST Sumacab Campus, Cababatuan Nueva Ecija.

4th year capstone project.

Functions

  • User login
  • 2 types of user registration - Students and Facilitator
  • Rate and Reviews
  • Facilitator can create, update and delete its own facility

Programming languages and Frameworks

  • Laravel
  • Bootstrap
  • Javascript, Jquery and Ajax
